# 🔎 Buscador de Perfis do GitHub

Aplicação web desenvolvida com HTML5, CSS3 e JavaScript que permite pesquisar perfis públicos do GitHub utilizando a API oficial da plataforma. O projeto exibe informações do usuário de forma dinâmica e intuitiva.

## ✨ Funcionalidades

- Pesquisa de perfis do GitHub
- Consumo da API oficial do GitHub
- Exibição de foto do perfil
- Exibição do nome e nome de usuário
- Informações públicas do perfil
- Tratamento de usuário não encontrado
- Interface simples e responsiva

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- GitHub REST API

## 📁 Estrutura do Projeto

```
Projeto1-Buscador-de-github/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
└── README.md
```

## 🚀 Como Executar

1. Clone este repositório.
2. Abra o arquivo `index.html` em seu navegador.
3. Digite o nome de um usuário do GitHub.
4. Visualize as informações retornadas pela API.
